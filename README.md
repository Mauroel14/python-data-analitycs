# Calculador de Huella de Carbono: Análisis de Movilidad Laboral 🌍🚗

## 📝 Descripción del Proyecto
Este proyecto fue desarrollado para medir y analizar el impacto ambiental de los traslados del personal hacia la empresa. El script procesa datos de geolocalización para calcular la **Huella de Carbono** 
individual basada en la distancia recorrida y el tipo de transporte utilizado (Auto, Moto, Transporte Público, etc.).

## 🚀 Funcionalidades Clave
* **Cálculo de Distancias:** Obtención de coordenadas exactas mediante API para medir el trayecto casa-trabajo.
* **Clasificación por Movilidad:** Diferenciación de emisiones según el medio de transporte.
* **Visualización de Impacto:** Gráficos que muestran la evolución mensual de las distancias recorridas y su correlación con la huella de carbono.

## 🛠️ Stack Tecnológico
* **Python:** Procesamiento de datos y lógica de cálculo.
* **APIs de Geolocalización:** Para la obtención de coordenadas precisas.
* **Pandas & Matplotlib:** Limpieza de datos y generación de reportes visuales.

## 📁 Archivos Principales
* `Cordenadas.py`: Conexión con la API y mapeo de puntos geográficos.
* `distancia_movilidades.py`: Algoritmo que calcula los kilómetros recorridos según el vehículo.
* `Gráfico de distancia recorrida.PNG`: Dashboard visual de los traslados analizados.

## 💡 Valor Agregado
Este indicador permite a las organizaciones tomar decisiones basadas en datos para promover políticas de movilidad sustentable y reducir su impacto ambiental operativo.
